# 🎮 Quiz Game

A simple Python-based Quiz Game that asks multiple-choice questions, checks user answers, and displays the final score.

## ✨ Features

- Multiple Quiz Questions
- User Input
- Instant Answer Checking
- Final Score Calculation
- Beginner-Friendly Python Project

## 🛠 Technologies Used

- Python
- Conditional Statements (if/else)
- Variables
- User Input
- Functions (Future Version)

## 📂 Project Structure

Quiz-Game/
│── main.py
│── questions.txt
│── requirements.txt
│── README.md

## ▶️ How to Run

1. Install Python.
2. Download or clone this repository.
3. Run the following command:

```bash
python main.py
```
